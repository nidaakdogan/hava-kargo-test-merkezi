🔧 Teknik İyileştirmeler

  • Sticky Header: Scroll'da shadow ve backdrop blur
  • 8-pt Grid: Yatay padding 16px, kart araları 12-16px
  • Typography: Başlık 18-20pt, bölüm 16-18pt, gövde 13-14pt
  • Line Height: Başlık %120-130, gövde %130-140
  • Touch Targets: Tüm dokunma alanları ≥44×44px
  • Accessibility: WCAG AA kontrast, aria-label'lar
  • Responsive: Mobilde sticky header kaldırıldı
  • Spacing: Chart başlık ile çubuk arası 12-16px
